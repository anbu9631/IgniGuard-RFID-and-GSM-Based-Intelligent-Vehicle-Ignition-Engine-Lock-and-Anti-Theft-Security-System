IgniGuard is an IoT-enabled vehicle security system designed to prevent unauthorized vehicle access and improve theft protection. It combines RFID authentication with GSM communication to provide secure ignition control, remote engine locking, and instant SMS notifications.

The system allows only registered RFID cards to start the vehicle. If unauthorized access is detected, the engine remains locked and an alert is immediately sent to the vehicle owner via GSM. Users can also remotely disable or enable the engine through SMS commands, adding an extra layer of security even when the vehicle is unattended.

Features:
🔐 RFID-based authorized vehicle ignition
📱 GSM-based remote engine lock/unlock
🚨 Instant SMS alerts for unauthorized access
🔑 Secure user authentication
⚡ Low-cost embedded security solution
🚗 Real-time anti-theft protection
🔧 Easy integration with vehicle ignition systems
Technologies Used:
Arduino/ESP32
RFID Reader (EM-18 Reader module)
GSM Module (SIM800L/SIM900A)
Embedded C
Relay Module
Power Supply Circuit
Applications:
Personal vehicles
Motorcycles
Fleet management
Commercial transportation
Smart vehicle security systems

This project demonstrates how embedded systems and IoT technologies can be integrated to create a reliable, affordable, and intelligent vehicle anti-theft solution.
